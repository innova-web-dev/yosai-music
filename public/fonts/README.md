# Monument Extended Font Files

Place your Monument Extended `.woff2` font files here:

- `MonumentExtended-Regular.woff2`
- `MonumentExtended-Bold.woff2`
- `MonumentExtended-Ultrabold.woff2`
- `MonumentExtended-Black.woff2`

These are loaded via `@font-face` in `src/app/globals.css`.

Monument Extended is a commercial typeface by [Pangram Pangram Foundry](https://pangrampangram.com/products/monument-extended).
